Написать при помощи JavaScript код страницы,

которая при загрузке проверяет разрешение экрана монитора

(атрибут screen.width возвращает разрешение по горизонтали)

и в зависимости от результата

(640х480, 800х600, 1024х768)

использует одну из 3 возможных таблиц стилей

(small.css, middle.css, large.css)
